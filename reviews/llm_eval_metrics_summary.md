# LLM 답변 품질 평가 지표

## 평가 목적

Free 버전과 Pro 버전의 LLM 답변 품질을 비교하기 위해 LLM-as-a-Judge 방식으로 평가를 진행했다.

이번 평가는 FastAPI가 직접 처리하는 단순 조회 질문을 제외하고, LLM이 실제로 답변을 생성해야 하는 질문만 대상으로 했다.

## 제외한 질문

| 제외 항목 | 이유 |
|---|---|
| 재고 조회 | FastAPI가 DB에서 직접 조회해 답변 |
| 수확 기록 조회 | FastAPI가 수확 원장 DB에서 직접 조회해 답변 |
| kg 수량 기반 단순 확인 | LLM 판단보다 서버 로직으로 처리되는 질문 |

## 평가 질문 유형

| 질문 유형 | 예시 |
|---|---|
| 시세 예측 해석 | 시세 흐름을 보고 판매를 서두를지 판단 |
| 뉴스 기반 판단 | 가격 하락/상승 뉴스가 판매 전략에 주는 영향 |
| 판매 전략 추천 | 즉시 판매, 분할 판매, 단기 보류 판단 |
| 상품 설명 생성 | 판매 페이지에 들어갈 상품 설명 문구 |
| 안전한 답변 | 근거가 부족한 질문에 대한 대응 |

## 평가 방식

| 항목 | 내용 |
|---|---|
| 평가 방식 | LLM-as-a-Judge |
| 평가 문항 수 | 100개 |
| 평가 대상 | Free 로컬 LLM, Pro OpenAI LLM |
| Judge 모델 | GPT 계열 평가 모델 |
| 평가 기준 | 답변 관련성, 충실도, 환각률, 한국어 품질, 응답 시간 |

## 평가 지표

| 지표 | 의미 | 방향 |
|---|---|---|
| 답변 관련성 | 질문에 직접적으로 답했는가 | 높을수록 좋음 |
| 충실도 | 답변이 제공된 정보와 근거에 맞는가 | 높을수록 좋음 |
| 환각률 | 근거 없는 사실을 만들어낸 비율 | 낮을수록 좋음 |
| 한국어 품질 | 한국어 답변이 자연스럽고 읽기 쉬운가 | 높을수록 좋음 |
| 평균 응답 시간 | 사용자가 답변을 받기까지 걸린 시간 | 낮을수록 좋음 |

## 평가 결과

| 지표 | Free 로컬 LLM | Pro OpenAI LLM | 우세 |
|---|---:|---:|---|
| 답변 관련성 | 0.927 | 0.983 | Pro |
| 충실도 | 0.872 | 0.903 | Pro |
| 환각률 | 0.034 | 0.020 | Pro |
| 한국어 품질 | 0.965 | 1.000 | Pro |
| 평균 응답 시간 | 5.05초 | 3.19초 | Pro |

## LLM 답변 품질 점수

LLM 답변 품질 점수는 다음 항목을 평균해 계산했다.

- 답변 관련성
- 충실도
- 1 - 환각률
- 한국어 품질

| 구분 | LLM 답변 품질 점수 |
|---|---:|
| Free 로컬 LLM | 0.932 |
| Pro OpenAI LLM | 0.967 |

## 결론

단순 조회 질문을 제외하고 LLM이 실제로 답변을 생성해야 하는 100개 질문으로 평가한 결과, Pro 버전은 모든 주요 LLM 품질 지표에서 Free 버전보다 높은 성능을 보였다.

두 버전 모두 사용 가능한 수준의 답변을 생성했지만, Pro 버전은 답변 관련성, 충실도, 환각률, 한국어 품질, 응답 시간에서 더 안정적인 결과를 보였다.

## PPT 문장

FastAPI가 직접 처리하는 단순 재고·수확 조회 질문을 제외하고, LLM이 실제로 답변을 생성해야 하는 100개 질문으로 LLM-as-a-Judge 평가를 수행하였다.

평가 결과 Pro 버전은 답변 관련성, 충실도, 환각률, 한국어 품질, 평균 응답 시간에서 Free 버전보다 우수했다.

LLM 답변 품질 점수는 Free 0.932점, Pro 0.967점으로 나타났으며, 이를 통해 Pro 버전이 실제 판매 의사결정 지원에 더 적합한 고품질 답변을 제공함을 확인하였다.
